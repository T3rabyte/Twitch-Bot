## Ontwerp

Deze bot is een intermediair tussen de live chat en de live stream Output.

De bot leest alle berichten die in de live chat gestuurd worden. Aan de hand van het bericht word er gekeken of de bericht een commando is dat de bot moet uitvoeren of een normaal bericht is en er niets mee hoeft te doen. Door de commando's die gestuurd worden kan de live chat een game spelen. In dit geval gaat het om de game: Pok√©mon Red. En is het aan de live chat en kijkers om het spel uit te spelen. Met de mogelijkheid om de Commando's uit te breiden voor het spelen van andere game's.

De commando's zijn:
- omhoog (Laat het karakter of de interface omhoog gaan)
- beneden (Laat het karakter of de interface omlaag gaan)
- links (Laat het karakter of de interface naar links gaan)
- rechts (Laat het karakter of de interface naar rechts gaan)
- aknop (drukt de Nintendo a knop in)
- bknop (drukt de Nintendo b knop in)
- start (drukt de Nintendo start knop in)
(Niet hoofdletter gevoelig)

Mijn speciale input is een live chat die de berichten daar in verwerkt.
